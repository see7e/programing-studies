Inline style

A configuração de estilo é aplicada dentro da tag, modificando de acordo com o atributo espeficicado. Não é recomendado pois torna a ligação entre as diversas possibilidades de configuração de esstilo mais difíceis e, dentro de um texto muito grande, praticamente impossível de perceber.

<p style="color: green;">

Internal style sheet

No caso da configuração de estilo interna, feita dentro da <head> usa-se seletores, que buscam tags específicas e modificam elas de forma simultânea.

<style type="text/css">
p {
color: blueviolet
}
