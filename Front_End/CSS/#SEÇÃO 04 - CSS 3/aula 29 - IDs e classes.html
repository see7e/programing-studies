Classes

A configuração de classes é aplicada para um grupo de elementos similares, utilizada como exceção, muito embora seja para mais de um único elemento.

<style type="text/class">
.paragrafo {color: green}
</style>
Dentro da tag: class="paragrafo"

ID (cabeçalho)

Utilizado uma única vez, para elementos específicos.

<style type="text/class">
#cabeçalho {background: lightgray}
</style>
Dentro da tag: id="secundário"
